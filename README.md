# 📐 Java Area Calculator

**Version:** v1.0 (Console Edition)

A simple and interactive console application that calculates the area of different geometric shapes. Built to practice Java's `switch-case` control flow, user input handling, and the built-in `Math` library.

### ⚙️ Core Features
* **Interactive Menu:** A numbered selection system (1-3) designed to prevent typo-related errors.
* **High Precision:** Utilizes `Math.PI` and `Math.pow()` for accurate mathematical results.
* **Error Handling:** Includes a `default` case to gracefully handle invalid user inputs.
* **Supported Shapes:** Circle, Square, and Triangle.

### 🚀 Upcoming Updates (v2.0)
* Adding support for more geometric shapes (e.g., Rectangle, Cylinder).
* Implementing a loop to allow multiple calculations without restarting the application.
